Imagic is a tool that can edit images using Go's standard library image package.

The base tool is just a CLI but a GUI version made with ebitengine is planned for the future.

Features:

- Rotate, flip and crop images.
- Gray scale.
- Blend images with a color.
- Change contrast and brightness of an image.
- Convert to ASCII.
- Blur an image.
- Draw images on top of other images.
